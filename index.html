Pace Project

To complete my degree I need to complete what is called a PACE unit. What this means is that I in a team of similar disciplines create a project with an industry contact. In my case, it was to develop an exploratory web application for uploading client data working with Faethm AI. This project allows a user to upload their data which then gets fed to the python backend for analysis using NumPy and Pandas. Then the user will assign the appropriate headers to their data which would then be sent to Faetham Ai.

Technologies used

Javascript, React, Python, Flask, NumPy, Pandas


D&D Helper

This application is a collection of tools that hopefully prove useful to players of Dungeons and Dragons. This web app contains: A tool for comparing different dice roll combinations by conducting Monte Carlos simulations to determine which of the dice rolls are most likely to be better. A name generator that creates names that conform to requirements for all of the races of the player's handbook.

Technologies used

Javascript, React, Material UI, Bootstrap, Express, Heroku

Twitter Clone

Over the course of my Bachelor's of IT degree, One of the assignments was to create a Twitter clone. The requirements for this assignment was for there to be functionality for login, registering and posting. Other requirements include having post-filtering based on whom a user is following, their likes and mentions.


Technologies used

Javascript, React, Express, Heroku

Order Management System

Over the course of acquiring my certificate IV Programming, one of the assignments was to create an Order Management System. This began with creating an SQL database, then creating a WPF application that connects to the database. Then the development of the functionality for creating orders, reading existing orders, updating existing orders and the deletion of orders.

Technologies used

C#, .NET, WPF, SQL
